## ✨ 简介
自动打开浏览器并打开某合集视频中的最新一期视频。可将bat脚本加入开机自启动，开机时自动打开最新一期视频。

## 🍨 教程 
方法1：直接用python运行
```python
python bilibili.py
``` 
  
方法2：将bat脚本加入开机自启动，开机时自动打开最新一期视频。  
加入开机自启动，快捷键win+R，输入以下命令打开的startup文件夹：  
```
shell:startup
```  
再将bat脚本加入打开的startup文件夹中。  
bat内的路径需修改为实际路径。默认执行当前文件加内的bilibil.py脚本  
此时就完成了，开机时会自动启动。
